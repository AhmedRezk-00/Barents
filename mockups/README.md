# BFP-BARENTS Mockup
Es gibt drei python-basierte standalone mockups.

## initial-mockup
Dieses Mockup ist ein erster Versuch, das Erstellen und Bearbeiten von RDF Instanzen der BARENTS Ontologie graphisch umzusetzen. 

Im Rahmen dieses Mockups wird UX, etc. ignoriert. Ausserdem wurde nicht auf clean code oder sonstige Richtlinien geachtet. 

## logic-mockup 
Dieses Mockup sollte Klarheit im Bezug auf die Logik des rdf managers bringe und wie dies mit einer visuellen Graphdarstellung zu vereinen ist.

## test-mockup
Ein kleines mockup bei dem mit rdflibs experiment wurde. 

## How-to
mockup.py ist jeweils das Entry file von dem das Program ausgefuehrt werden kann.
